Global Operations- Sprite Effects Descriptions





Fire (with smoke and sparks)

Sprite Effect ID: 158


Info: Adding this effect will also spawn a black smoke emitter, constant light effect, and a rising spark effect. Designed for oil barrels, but works in many other scenarios.


Fire (without smoke and sparks: better on fps)

Sprite Effect ID: 395


Info: Generic fire. If you plan on having more than a few fire emissions in your level, use this one. It’s much better on the framerate.


More Fire (without smoke and sparks: better on fps)

Sprite Effect ID: 254


Info: Similar to the one above.


Smoke: Chimney

Sprite Effect ID: 340


Info: Used on the chimney in Chechnya. A slow, black smoke emission.


Frosty Steam Vent

Sprite Effect ID: 380


Info: Used on the vents in Antarctica. A slow moving, white steam.


Smoke: Argentina Server

Sprite Effect ID: 307


Info: A black smoke that gets emitted from the Argentina Security system after it gets destroyed.


Smoke: White

Sprite Effect ID: 258


Info: A generic white smoke.


Sparks

Sprite Effect ID: 84 (emits every 15 seconds)

Sprite Effect ID: 85 (emits every 23 seconds)


Info: Sparks that emit in bursts every so often. Used on the neon sign in Mexico.


Water Cooler Bubbles

Sprite Effect ID: 165


Info: Bubble emitter, made for water coolers. Make sure the rotation is set to point up.


Halo: Quebec

Sprite Effect ID: 351


Info: The Halo effect that is used on some lights in the Quebec level.


Halo: South China Sea

Sprite Effect ID: 28


Info: The Halo effect that is used on some lights in the South China Sea level.


Halo: Lima

Sprite Effect ID: 326


Info: A Halo effect that is used in Lima. Smaller than the Quebec and Colombian halos.


Halo: Colombia

Sprite Effect ID: 373


Info: A Halo effect that is used in Lima.


Halo: Red Light

Sprite Effect ID: 44


Info: A Halo effect that is used on some red lights on the game.


Halo: Blinking red C4 light halo (with beeping sound)

Sprite Effect ID: 53


Info: A Halo effect that is used on the big C4 bomb. It also plays a beeping sound.


Spotlight

Sprite Effect ID: 55


Info: A light with a halo, and a beam that projects outwards. Used in South China Sea and Argentina in a few places.


Blowing Snow

Sprite Effect ID: 57


Info: Snow that lightly drifts across the ground. Used outside, in Antarctica.

Constant Light

Sprite Effect ID: 87


Info: A Reddish-Orange dynamic light that can be added to fire effects, etc…


Flashlight

Sprite Effect ID: 170


Info: Effect that is used when a player turns on his Tack-Light in the game.


Streetlight

Sprite Effect ID: 386


Info: Used in some of the Training levels. A Halo and beam of light that come off of the street lamps.


Streetlight: Tunnel Level

Sprite Effect ID: 391


Info: Same thing as above, but a variation that was used in the Tunnel Level.


Waterfall Mist

Sprite Effect ID: 284 (small)

Sprite Effect ID: 283 (medium)

Sprite Effect ID: 202 (large)

Sprite Effect ID: 285 (Jumbo)

Sprite Effect ID: 290 (Used in USA)


Info: Used on the waterfalls in Sri Lanka and USA


Candle Flame

Sprite Effect ID: 417


Info: A flame effect that is used on the candles in Mexico.


Steam: Training Level Vents

Sprite Effect ID: 430


Info: Used on big steam vents in some of the Training Levels. Big effect. Not recommended for Multiplayer maps, due to the framerate hit.
